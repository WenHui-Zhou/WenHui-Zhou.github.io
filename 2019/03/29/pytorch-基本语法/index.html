torch 导入数据的pipline（流程）

torchvision是torch中一个用于 生成图片，数据集，模型类，欲训练模型的包。它主要包含一下几个部分：

  1. torchvision.datasets: 用于导入一些比较流行的开源数据集（cifar等）
  2. torchvision.models: 包含了很多流行的网络框架，包括alexnet，VGG，resnet，以及一下欲训练模型
  3. torchvision.transforms: 定义了一些常用的数据预处理的函数，如random crop，rotate等等
  4. torchvision.utils: 里头定义了很多好用的函数，如保存图片等
