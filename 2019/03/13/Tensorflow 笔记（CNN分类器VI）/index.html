CNN分类网络

CNN网络在原来网络的基础上加入了卷积层，对特征进行提取后分类，能够提升网络的分类准确率，同时在网络中加入了dropout，缓解网络的过拟合。

写一个分类器的基本思路如下：

Assemble graph:

  1. read data
  2. create placeholder
  3. create weight and bias in a layer
  4. create a net structure
  5. specify loss function
  6. create optimizer

train model:

  1. specify the epochs,iteration,batch-size
  2. initial variables
  3. step by step run the optimizer（use feed_dict to feed data into x,y placeholder）
